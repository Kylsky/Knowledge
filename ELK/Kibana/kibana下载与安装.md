# kibana下载与安装

## 一、下载安装包

https://www.elastic.co/cn/products/kibana

```
curl -O https://artifacts.elastic.co/downloads/kibana/kibana-7.6.2-linux-x86_64.tar.gz
```



## 二、解压

```sh
tar -xzf kibana-7.6.2-linux-x86_64.tar.gz
cd kibana-7.6.2-linux-x86_64/
```
